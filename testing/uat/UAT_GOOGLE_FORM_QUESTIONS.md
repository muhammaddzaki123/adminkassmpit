## 📊 SCORING SYSTEM

### Untuk Rating Scale (1-5):
- **1-2:** Tidak Puas/Perlu Perbaikan Serius
- **3:** Cukup/Acceptable
- **4-5:** Puas/Sangat Puas

### Hitung Score Keseluruhan:
```
Total Rating Questions: 30 pertanyaan rating (1-5)
Maksimal Score: 150
Scoring:
- 120-150: Excellent (80-100%)
- 100-119: Good (67-79%)
- 80-99: Fair (53-66%)
- 60-79: Poor (40-52%)
- <60: Very Poor (<40%)
```

## 💡 TIPS IMPLEMENTASI

### 1. Conditional Logic
Set conditional display berdasarkan role:
- Jika pilih "Admin" → tampilkan pertanyaan Admin
- Jika pilih "Bendahara" → tampilkan pertanyaan Bendahara
- Dst.

### 2. Required vs Optional
- **Required:** Pertanyaan 1-35 (wajib dijawab)
- **Optional:** Pertanyaan 36-50 (opsional)

### 3. Validasi Input
- Email: Gunakan "email" type untuk validasi format
- Rating scale: Set 1-5 sebagai opsi
- Checkbox: Izinkan multiple selection

### 4. Customization Appearance
- Logo sekolah di header
- Warna sesuai brand
- Intro message yang friendly
- Closing message "Terima kasih atas feedback Anda"

### 5. Response Options
- Batasi 1 tanggapan per email (jika ingin)
- Tampilkan hasil real-time untuk progress tracking
- Send email confirmation ke responden

### 6. Distribution
- Buat link pendek
- Share via:
  - Email ke seluruh pengguna
  - WhatsApp group
  - Dashboard sistem (popup notification)
  - Cetakan QR code (untuk user yang kurang tech-savvy)
